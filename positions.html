By default the value for position property is static.
.relative-sub-box2{position: relative;top: -20px;left: -10px;}
relative element can be moved from its default position using top,right,bottom and left css properties
position:absolute will take the element out of order and places it seperate. Other static elements will go under it. We can control the position with Top, Right, Bottom, Left. We need to specify width, or the element will become inline element
last absolute element will be on the top of the other absolute elements. Box 1 comes after this element, according to the code written. But we can control that using z-index. Higher the value, on the top will be the element.
If the parent element is a static element, then the absolute elements will consider body as its parent and placed according to the Top,Right,Bottom and Left values. We have to make its parent element other than static positioned, in order to keep the absolute element in its parent
